Work Stopwatch

2022-06-24

Malcom Gladwell’s assertion that it takes 10,000 hours to become world-class at any technical skill has stuck with me more than any other conclusion from Outliers. Mostly as an excuse to learn Tkinter and practice reading and writing files in Python, I’ve built a small Work Stopwatch that tracks how long you’ve spent on a specific task. Soon you, too, will be able to track your progress to the golden 10,000-hour number! The code is here on Github.

In the process of working on this I also learned that pyinstaller works well to package Python files as executables, though the -onefile flag is a necessity if you plan on sharing these executables with others.
